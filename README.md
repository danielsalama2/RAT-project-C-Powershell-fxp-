2) לשנות את הקישור שנמצא בקוד של הclient side לקישור של האתר שלכם
3) להדביק באתר שלכם קוד של  הreverse shell שלכם עם האייפי שלכם (הקוד נמצא בגיטהאב)
4) להכנס לserver side ולהתחיל להאזין על הפורט
